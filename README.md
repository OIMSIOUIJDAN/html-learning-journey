1) Text Formatting in HTML

HTML provides tags to format text and emphasize meaning.

1.1 Bold Text
<b>This text is bold</b>

1.2 Strong Text (Recommended)
<strong>This text is important</strong>

1.3 Italic Text
<i>This text is italic</i>

1.4 Emphasis (Recommended)
<em>This text is emphasized</em>

1.5 Underlined Text
<u>This text is underlined</u>

1.6 Highlighted Text
<mark>This text is highlighted</mark>

2) Line Breaks and Horizontal Lines
2.1 Line Break

The <br> tag moves text to the next line.

Hello<br>
World

2.2 Horizontal Line

The <hr> tag creates a horizontal line.

<hr>

3) Lists in HTML

Lists help organize information clearly.

3.1 Unordered List (Bullets)
<ul>
  <li>Milk</li>
  <li>Eggs</li>
  <li>Bread</li>
</ul>

3.2 Ordered List (Numbers)
<ol>
  <li>Wake up</li>
  <li>Study</li>
  <li>Sleep</li>
</ol>

4) Links in HTML

Links allow users to open other webpages.

4.1 Basic Link
<a href="https://www.google.com">Visit Google</a>

4.2 Open Link in a New Tab
<a href="https://www.google.com" target="_blank">Visit Google</a>

5) Images in HTML

Images are added using the <img> tag.

5.1 Basic Image
<img src="cat.jpg" alt="A cute cat">

5.2 Image from a Website
<img src="https://example.com/image.jpg" alt="Example image">

Notes

src is the image path or URL

alt describes the image if it fails to load
